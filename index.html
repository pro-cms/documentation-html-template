Getting Started

Smart-AI is a laravel based application which can be used for those who wants to start copywriting business or provide AI service to other businesses using GPT-3 models by Open AI. With the most popular payment gateways ie paypal and stripe you can easyly charge your customers using the token system just like the one with Open AI.

How Does It Works?

The system working principle is very simple, First A customer registers an account and then login in to the panel, to use the system API or copywriting , one have to buy token in the client area where 1 token is approximately 4 characters or 0.75 words., the price per each token is set by admin in the admin panel.
Now The user can make API calls and use the system to copywrite his business or to provide AI service to other businesses.

Application based system

To ensure reports are kept and your client can be separeted, we have introduced apps system, suppose your client have two project, FOODAI and FRIENDSAI, we terms both of this as apps and report is based on this applications.

How to Install Smart AI

You can install our script in both shared and other types of hosting without any problem

Follow Below steps

  1. Upload the content of the script files to public_html/
  2. Unzip the content in your installation directory
  3. Create Database and Mark Database username, password and host
  4. Head to .env file in the script located in core/.env
  5. Change the database details
  6. Update other information of your app eg, APP_NAME
  7. Update stripe and Paypal keys
  8. Head to Phpmyadmin and upload the database file located in assets/database.sql
  9. Your Application is ready
  10. admin link: htps://app_url.example/admin

Get Open AI API key

How to get Open AI API key

Head to https://beta.openai.com/login/ and login with your email and password. Go to Keys section and get/create API key.
Then head to your application admin area > configuration > API section and update the keys.
